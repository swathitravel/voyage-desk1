VOYAGE DESK — NO NODE.JS REQUIRED

This folder is ready for GitHub Pages. It already contains your Firebase web configuration.

Upload these files directly to the root of a GitHub repository:
- index.html
- styles.css
- app.js
- manifest.webmanifest
- icon.svg
- README.txt

Then use GitHub Pages:
Settings > Pages > Deploy from a branch > main > /(root)

After GitHub gives you the site address, add YOUR-USERNAME.github.io to:
Firebase Authentication > Settings > Authorized domains

The app uses Google sign-in and Cloud Firestore to synchronize data across devices.
